# Reading List — Agent Instructions

Filename format: `YYYY-author-short-topic.pdf` (`0000` if undated).

`INDEX.md` is the master index. Entry format:

```
- [filename](filename)
  - one-line description
```

On new paper: read page 1-2 for year/author/topic, rename to convention, add to correct `INDEX.md` section (create section if needed, keep alphabetical). On removal: delete entry from index.
